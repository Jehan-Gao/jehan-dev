# Jehan's Personal Website

My personal website: https://jehan-dev.com — built with the [Astro](https://astro.build) stack.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Sync Strava Activities

```bash
pnpm sync:strava:activities
```

## Deploy to GitHub Pages

The site is automatically deployed to GitHub Pages when pushing to the `main` branch.

## Custom Domain

The site uses the custom domain: https://jehan-dev.com

## Tech Stack

- **Framework**: Astro
- **Styling**: Tailwind CSS
- **Map**: Leaflet with CartoDB/AMap
- **Data Sync**: Strava API
